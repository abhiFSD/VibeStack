Now let's get started with the Value Stream Mapping report.

The Value Stream Mapping report has a comprehensive visual interface designed to map your entire value stream. At the top, you'll see the Action Items section for tracking improvement opportunities.

Below that, you'll find the main VSM interface with several key components:

SUMMARY DATA - will be auto-generated as you add data. For the Total Lead Time, Total Cycle Time, and Total Wait Time or Inventory, you can change the time unit.

TOTAL DEMAND/TARGET DATA - you can add Total Demand/Volume and Time to Produce and save, Takt Time/Target will be calculated.

INFORMATION FLOW and KAIZEN PROJECT is a simple text area to add more details.
