## Tool 12: Waste Walk

Now let's get started with the Waste Walk report. 

The Waste Walk report has a unique interface. At the top, you'll see two tabs - "HIGHLIGHTS REPORT" which is selected by default, and "Departments".

Similar to A3, DMAIC, and PDCA reports, Waste Walk reports use the HIGHLIGHTS REPORT format which is specifically designed for documenting waste identification and elimination opportunities found during your walk.

You can add action items separately for tracking specific waste elimination initiatives.

A Waste Walk is a systematic observation of your workplace to identify the 8 wastes of lean:
- Defects: Errors requiring rework or scrap
- Overproduction: Making more than customer demand
- Waiting: Idle time for people or equipment
- Non-utilized talent: Not using people's skills fully
- Transportation: Unnecessary movement of materials
- Inventory: Excess materials or WIP
- Motion: Unnecessary movement of people
- Extra processing: Doing more than customer requires

In the HIGHLIGHTS REPORT view, you can create structured sections for each waste type or area observed. Each highlight can focus on specific waste findings.

To add a new section, click the Add button. You can then:
- Enter a title for your waste observation
- Select the waste type from the 8 wastes
- Add detailed description using the rich text editor
- Upload photos of the waste observed
- Assign team members responsible for addressing the waste

The highlights are displayed in a grid layout, making it easy to view all your waste observations on one screen.

Finally, you can export to PDF and get yourself a comprehensive Waste Walk report documenting all observations, photos, and improvement opportunities.

The PDF export provides a professional format perfect for sharing findings with leadership and planning waste elimination kaizen events.

That's it for the Waste Walk report. It's an excellent tool for systematic waste identification and elimination. Hope you like this, and if you see any issues or face any difficulties, please contact us using this information, and we'll get in touch with you in no time.

## Tool 13: 5 Whys

Now let's get started with the 5 Whys report.

Now, this is the Action Item section where you can create action items for this report.

The 5 Whys report uses a unique system for root cause analysis. Notice that instead of "Categories", the system uses "Problems" for this report type.

To add a problem, click the plus button. Enter the name of the problem you want to analyze. Unlike other reports, 5 Whys doesn't have default categories - you'll create custom problems specific to your investigation.

Each problem can include:
- Problem statement and root cause analysis chain
- Additional details with descriptions
- Attachments like photos, documents, and evidence
- Assignees responsible for the investigation

When you click on a problem card to add statements, notice that they're called "Whys" instead of "Statements" for this report type.

For each Why, you can:
- Enter the question (Why does this happen?)
- Document the answer/finding
- Add supporting evidence
- Assign severity or priority values

In the Additional Details section for each problem, you can:
- Add comprehensive problem descriptions
- Upload photos of the issue
- Document impact and urgency
- Tag relevant team members

You can rearrange problems using the rearrange button to organize them by priority or investigation sequence.

## Tool 18: Pareto Chart

Now let's get started with the Pareto Chart report.

The Pareto Chart report implements the 80/20 principle, helping you identify the most significant factors contributing to a problem or situation.

At the top, you'll see the Action Items section for tracking improvement initiatives based on your Pareto analysis.

**Data Entry Interface**
Below that, you'll find a data table for inputting your problem categories and their frequencies:

Click "Add" to create new data entries. For each category, you'll provide:
- **Label**: The category or problem type (e.g., "Defect Type A", "Customer Complaint Category")
- **Value**: The frequency, count, or cost associated with this category
- **Description**: Additional details about this category

**Data Management**
The system automatically sorts your data by value in descending order (highest to lowest), which is essential for proper Pareto analysis.

You can:
- Edit any entry by clicking the edit icon
- Delete entries using the trash icon
- Add as many categories as needed for comprehensive analysis

**Pareto Chart Visualization**
The system generates a combined bar and line chart showing:
- **Bars**: Individual category frequencies (sorted from highest to lowest)
- **Line**: Cumulative percentage across all categories
- **80/20 Reference**: Visual indication of the critical few vs. trivial many

**80/20 Analysis**
The Pareto Chart helps you identify:
- **Vital Few**: The small number of categories causing most problems (typically 20% of causes creating 80% of effects)
- **Trivial Many**: The larger number of categories with less impact
- **Priority Focus**: Where to concentrate improvement efforts for maximum impact

Export your Pareto analysis to PDF, including:
- Complete data table with frequencies and percentages
- Visual Pareto chart with both bars and cumulative line
- 80/20 analysis summary and key insights
- Recommended actions for high-impact categories

The PDF provides professional documentation for management reviews, improvement planning, and resource allocation decisions.

That's it for the Pareto Chart report. It's a powerful tool for prioritizing improvement efforts based on data-driven analysis. Hope you like this, and if you see any issues or face any difficulties, please contact us using this information, and we'll get in touch with you in no time.
